#  Deep Learning Projects

A collection of hands-on Deep Learning projects built using **Python**, **TensorFlow**, and **Keras** — covering Artificial Neural Networks, Convolutional Neural Networks, and Recurrent Neural Networks through real-world datasets and Jupyter Notebook implementations.

---

## 📁 Projects

| Notebook | Description | Dataset | Algorithm |
|---|---|---|---|
| `ANN_Classification.ipynb` | Classifies 7 varieties of dates (Berhi, Deglet, Dokol, Iraqi, Rotana, Safavi, Sogay) using an ANN trained on 35 morphological features across 898 samples | Date Fruit Dataset | ANN |
| `ANN_Regression (1).ipynb` | Predicts continuous output values using a regression-based Artificial Neural Network — covers preprocessing, model building, and evaluation | Regression Dataset | ANN |
| `CNN_for_CIFAR10.ipynb` | Builds a Convolutional Neural Network to classify images across 10 categories from the CIFAR-10 benchmark dataset | CIFAR-10 | CNN |
| `RNN_for_sentimentanalysis.ipynb` | Performs sentiment analysis on text using a Recurrent Neural Network / LSTM — classifies text as positive or negative | NLP Dataset | RNN / LSTM |
| `Neuron.ipynb` | Explores how a single artificial neuron works — covers weights, bias, activation functions, and forward pass from scratch | — | Fundamentals |

---

## 🛠️ Tech Stack

- **Language:** Python 3
- **Notebooks:** Jupyter Notebook
- **Frameworks:** TensorFlow, Keras
- **Libraries:** NumPy, pandas, Matplotlib, scikit-learn

---

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/jeevan-G111/Deep_Learning_projects.git
   cd Deep_Learning_projects
   ```

2. **Install dependencies**
   ```bash
   pip install tensorflow keras numpy pandas matplotlib scikit-learn jupyter
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. Open any `.ipynb` file from the browser and run the cells.
